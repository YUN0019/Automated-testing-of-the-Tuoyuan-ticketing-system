🤖 自動化測試實驗室 (QA Portfolio Demo)
這是一個記錄我對 Python 自動化測試與網頁互動研究的專案。專案初衷是透過程式模擬瀏覽器行為，並自動產出測試報告。

專案內容
核心邏輯 (main.py): 主程式入口。負責自動啟動瀏覽器、執行斷言、存取快照並產出 Log。

報告系統 (test_reporter.py): 這是我寫的一個小工具，測試執行結束後，會自動將過程統計成一份 HTML 報表。

設定檔 (config.py): 統一管理網址與測試參數，方便快速調整。

儲存空間 (artifacts/): 存放測試過程中產出的錯誤快照或成功截圖。

執行方式
PowerShell
# 安裝相依套件
pip install -r requirements.txt

# 執行測試
python main.py
報表產出機制
這套自動化流程的亮點在於「交卷機制」：

程式在執行每一步時，都會透過 test_reporter.py 記錄成功或失敗的狀態。

執行結束後，會自動寫入一個 .html 檔案，裡面包含測試時間、成功率以及錯誤截圖連結。

直接打開該 HTML 檔即可看到視覺化的測試總結。
